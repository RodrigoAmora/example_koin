# example_koin
Descrição:
----------
Projeto de exemplo e aprendizado de Koin, a biblioteca de Injeção de Dependência do Kotlin. <br>
O projeto roda a partir do Android 7.0 (API Level 24)

<hr>

Dependências:
-------------
O projeto usa as seguintes dependências:
* Material Design
* Constraint Layout
* Koin
* Room
* ViewModel
